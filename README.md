CustomChat \ control your server chat.

============

Features 
Commands :
  - [ ] Mute mute player from chat
  - [ ] UnMute mute player from chat
  - [ ] Mute all player from chat
  - [ ] Unmute all player from chat
  - [ ] defPrefix default all from player
  - [ ] setPrefix from player
  - [ ] Nick from player
  - [ ] Chat Format Custom
    - [ ] Tag {WORLD_NAME}
    - [ ] Counter death
    - [ ] Counter murder
